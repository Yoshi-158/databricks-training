## 🔍 What Day 1 Demonstrates

### DAY1 - SQL Basic Query Practice
- Creates relational tables such as **Department**, **Employee**, and **Project**
- Inserts sample records for practical querying
- Uses `SELECT` statements for full and partial column retrieval
- Applies `WHERE` conditions for filtering records
- Uses `LIKE` for text-based pattern matching
- Performs date-based filtering using `YEAR()`, `MONTH()`, and date comparison
- Works with realistic employee-management sample data
